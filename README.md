# 🔐 RajMart Login Page

A sleek, modern and responsive login page crafted with pure HTML, CSS, and JavaScript. It features an animated gradient background, glassmorphism card design, password visibility toggle, and social login buttons — perfect for e-commerce or any web application frontend.

![RajMart Login Screenshot](https://via.placeholder.com/800x500?text=RajMart+Login+Preview)

## 🌟 Features

- **Glassmorphism UI** – Semi-transparent card with backdrop blur for a trendy look.
- **Animated Background** – Floating blurred orbs with smooth keyframe animation.
- **Password Toggle** – Show/Hide password with one click.
- **Remember Me & Forgot Password** – Basic login form utilities.
- **Social Login Buttons** – Ready-to-use Google and GitHub buttons (static).
- **Fully Responsive** – Optimised for mobile, tablet, and desktop screens.
- **No Dependencies** – Built with vanilla HTML, CSS, and JavaScript.

## 🛠️ Tech Stack

| Technology | Purpose |
|-----------|---------|
| HTML5     | Structure |
| CSS3      | Styling, animations, responsiveness |
| JavaScript (Vanilla) | Password toggle interactivity |
